# Инструкция по запуску Telegram-бота "Киркоров"

Этот файл содержит пошаговую инструкцию для запуска вашего Telegram-бота.

## 1. Подготовка

### 1.1. Получение токена бота

Если вы еще не получили токен для вашего бота:
1.  Откройте Telegram и найдите `@BotFather`.
2.  Начните чат с `@BotFather` и отправьте команду `/newbot`.
3.  Следуйте инструкциям, чтобы выбрать имя для вашего бота (например, "Киркоров") и имя пользователя (например, "KirkorovHelperBot").
4.  `BotFather` предоставит вам HTTP API токен. Он выглядит примерно так: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`. **Храните этот токен в секрете!**

### 1.2. Обновление токена в `bot.py`

1.  Откройте файл `bot.py` в каталоге `kirkorov_bot`.
2.  Найдите строку, которая выглядит так:
    ```python
    BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
    ```
3.  Замените `"ВАШ_ТОКЕН_ЗДЕСЬ"` на фактический токен, который вы получили от `@BotFather`. Убедитесь, что токен находится внутри двойных кавычек, например:
    ```python
    BOT_TOKEN = "8454130806:AAHT4Cx2i0KGcSAHrEGPZkPTmcCWynihA-I"
    ```
4.  Сохраните изменения в файле `bot.py`.

### 1.3. Установка Python 3.12 (если не установлен)

**Важно:** Этот бот лучше всего работает с Python версии 3.11 или 3.12, так как с Python 3.14 могут возникнуть проблемы с асинхронным циклом событий.

*   **Если вы используете macOS и у вас установлен Homebrew:**
    Откройте терминал и выполните:
    ```bash
    brew install python@3.12
    ```
*   **Если вы не используете Homebrew или у вас другая ОС:**
    Вам нужно будет загрузить и установить Python 3.12 вручную с официального сайта Python (python.org) или использовать другой менеджер версий Python (например, `pyenv`).

## 2. Настройка виртуальной среды

Виртуальная среда изолирует зависимости вашего бота от других проектов Python.

1.  **Перейдите в каталог бота в терминале:**
    ```bash
    cd /Users/aleksei/Documents/coding/Kirkorov/kirkorov_bot
    ```

2.  **Создайте виртуальную среду с Python 3.12:**
    Сначала найдите путь к исполняемому файлу Python 3.12, выполнив:
    ```bash
    which python3.12
    ```
    Затем используйте этот путь для создания виртуальной среды. Например, если `which python3.12` выдал `/usr/local/opt/python@3.12/bin/python3.12`, выполните:
    ```bash
    /usr/local/opt/python@3.12/bin/python3.12 -m venv venv
    ```
    (Замените `/usr/local/opt/python@3.12/bin/python3.12` на фактический путь, который вы получили).

3.  **Активируйте виртуальную среду:**
    ```bash
    source venv/bin/activate
    ```
    В вашей командной строке должно появиться `(venv)`, указывая на активную виртуальную среду.

4.  **Установите необходимые библиотеки:**
    ```bash
    pip install -r requirements.txt
    ```

## 3. Запуск бота

После выполнения всех подготовительных шагов:

1.  Убедитесь, что вы находитесь в каталоге `kirkorov_bot`.
2.  Убедитесь, что виртуальная среда активирована (`(venv)` в командной строке).
3.  Запустите бота следующей командой:
    ```bash
    python3 bot.py
    ```

Бот начнет работать и слушать сообщения в Telegram. Если ошибок не появилось, все работает!
